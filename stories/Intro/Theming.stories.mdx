# Theming

Import styled-components's [ThemeProvider](https://www.styled-components.com/docs/advanced#theming) in the root of your application and pass the theme.

```javascript
import React from 'react';
import { ThemeProvider } from 'styled-components';

const theme = {
  palette: {
    primary: '#ff0044',
    secondary: '#ff80a1',
  },
};

const App = props => {
  return <ThemeProvider theme={theme}>{/* Your code here */}</ThemeProvider>;
};

export default App;
```

> You don't need to set all props in your theme. It will be deep merged with the default theme
(except arrays).

## Variants

The `variant` prop that some components support is a combination of the `palette`, `colors` and
`grays`

### Palette

<Grid gridTemplateColumns="repeat(3, 1fr)" gridGap={15}>
  {Object.keys(palette).map(d => {
    const color = palette[d];
    return (
      <React.Fragment key={d}>
        <Box border="1px solid #ccc" p={10}>
          <Box bg={color} mb={2} width="100%" height={60} />
          <Paragraph>
            {d} - {color}
          </Paragraph>
        </Box>
      </React.Fragment>
    );
  })}
</Grid>
